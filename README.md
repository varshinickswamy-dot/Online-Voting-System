# Online-Voting-System
The Online Voting System is a web-based application designed to conduct elections digitally in a secure and efficient manner. It allows registered users to vote online while enabling administrators to manage candidates, monitor voting activity, and view results in real time.
The Online Voting System is a digital platform that allows users to cast votes securely through an online interface. It eliminates the need for physical polling stations and ensures a transparent, efficient, and user-friendly voting process.

This system includes authentication, candidate management, vote casting, and automatic result calculation.

Features

User registration and login authentication

Secure voting mechanism

One user can vote only once

Candidate management by admin

Real-time vote counting

Admin dashboard to monitor elections

Simple and user-friendly interface

How to Run the Project
1. Clone the Repository
git clone (https://github.com/varshinickswamy-dot/Online-Voting-System.git)
2. Navigate to Project Folder
cd online-voting-system
3. Install Dependencies
pip install -r requirements.txt
4. Run the Application
python app.py
5. Open in Browser
http://localhost:5000
